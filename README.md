
  # Portfolio Design

  Welcome to my portfolio repository! This project showcases my skills through a modern, responsive design. It includes examples of my recent projects.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Project Structure

  portfolio-design/
  ├── public/          # Static assets
  ├── src/
  │   ├── components/  # Reusable UI components
  │   ├── pages/       # Page-level components
  │   ├── styles/      # Global styles
  │   └── utils/       # Helper functions
  └── README.md
